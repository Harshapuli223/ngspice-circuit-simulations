# RC Circuit Simulation (Ngspice)

## Description

This project demonstrates the transient response of an RC circuit using Ngspice.

## Circuit Components

* Resistor: 1kΩ
* Capacitor: 1µF
* Input: Piecewise Linear (PWL) signal

## Simulation Type

* Transient Analysis (.tran)

## Observations

* Output voltage shows exponential rise due to capacitor charging.
* Delay in response compared to input signal.

## Files

* rc.cir → Ngspice netlist
* output.png → Simulation waveform

## Skills Demonstrated

* Netlist writing
* Transient simulation
* Waveform analysis

## Learning Outcome

Understanding of RC time constant and exponential charging behavior.
