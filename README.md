# Ngspice Circuit Simulations

This repository contains circuit simulation projects implemented using Ngspice, focusing on analog and digital circuit analysis for engineering and research applications.

## 🔧 Projects Included

* RC Circuit (Transient Response Analysis)
* CMOS Inverter (Voltage Transfer Characteristics & Switching)
* 6T SRAM Cell (Read/Write & Stability Analysis)

## 🧠 Skills Demonstrated

* Netlist-based circuit design
* Transient (.tran), DC (.dc), and AC (.ac) analysis
* Parameter sweeping and optimization
* Debugging simulation and convergence issues

## 🛠 Tools Used

* Ngspice
* LTspice (for visualization)

## 📌 Objective

To demonstrate strong fundamentals in circuit simulation and preparation for roles involving electronic simulation, VLSI, and AI-integrated hardware workflows.
